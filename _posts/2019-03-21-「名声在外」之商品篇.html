「名声在外」之商品篇

原创 花火田丁 花火田丁
花火田丁

huahuoding

对与错，是与非，下一秒钟你又在哪里？这里有一切关于「relativity」的故事

{{page.date}}

题图来自于NextDay（昨日的文）

不久前跟朋友说着买国外运动用品，近点儿日本亚马逊，远点儿美国亚马逊、德国亚马逊什么的，想来想去都是国外的渠道，但其实国内也能买到。

以前海淘，多半是因为国内价格虚高得太离谱，海淘就算加上运费、税费，也比国内价格低了不少，而且操作起来也不是很繁琐，算是某种程度上的“价廉物美”。

可不知从何时起，考虑海淘的主要因素不再是「价格」了，而是「正品」，并非国内渠道都是假货，但只要是经过了几次假货事件 ，这名声在外，心有戚戚。为什么婴幼儿用品的代购那么红火，也是因为某些品牌带来的“名声”，让孩子的爸妈放心不下。

名声在外，真是一个可怕的事儿，失去的信心也许花费当初几十倍几百倍的努力都无从挽回了。名声，或者换个词，「口碑」，高高立起，需要付出浩大的资源，半点儿差错就不能有。 可这高楼，轰然倒塌，却连一刻的功夫都不需要。

今天还在和同事讨论，与 B 端用户相比，C 端用户对于品质是相当挑剔的，不是说 B 端用户不挑剔，而是很多情况下，B端的采购方并不是直接的使用者，也许使用者也挑剔，但考虑到很多因素，最后很多“挑剔”都变成了悬而未决。但 C 端就不一样了，基本上采购者和使用者是同一个人，自己用的东西当然得讲究，花的是自己的钱，搭进去的是自己的时间呢。

因此 C 端产品的品控非常重要，「口碑」是把双刃剑，能把一个产品捧上天，也能让它落入谷底。对于 B 端产品，说实话，总避免不了修修改改，这其中服务提供方有很多机会去展现实力或者弥补过错。但放到 C 端，残酷的是，几乎只有1次机会，如果不乏替代品，估计只有 0 次机会，也就是说，如果不在很短时间或者就这一次的机会去展现正面的实力，那以后就很难去抓住用户的心了，而若是这次宝贵的机会中，还好巧不巧犯了错，那估计就从此就被打入冷宫了。

所以对于 C 端用户来说，如果暂时找不到替代品，那么无功无过的产品也能活下来，反过来，否则眼里是揉不进沙子。我们用一个 App 时也会去在意体验如何是否有用，何况是吃的穿的。

从民族文化传承的角度来说，理应是支持国货，但前提也是无论国货还是外国货，都要给消费者一定的信心才行，质量过关，诚实守信，那自然是最好的。可一旦伤了人心，再要挽回就比登天还难了。

这让我想起了某个实验，忘记了是什么小动物，大概场景是一直在一个地方设卡，让它没办法出去，久而久之，即便那地方打开了，竟然也没出去。人也是如此，一旦形成了习惯，或者脑中留下了印象，再要改观就要费很大的劲了。

名声攒起来难，毁起来可容易。
